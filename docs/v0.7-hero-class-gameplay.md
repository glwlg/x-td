# v0.7 职业玩法开发记录

## 本次目标

继续补齐完整玩法里的职业系统，让“不同职业有不同卡组、不同增益/限制、不同玩法占比”进入实际可玩版本，而不只停留在文档设定。

## 已完成功能

### 1. 开局职业选择

标题界面现在提供 3 个职业入口：

- `边境指挥官`：士兵推进 / 战术士气。
- `万灵召使`：建筑生产 / 兵潮召唤。
- `雷火方士`：法术爆发 / 控场节奏。

点击职业卡片会直接以该职业开始探索。

### 2. 职业初始卡组

三个职业拥有不同初始卡组：

- `边境指挥官`：保留原有均衡卡组，覆盖兵营、远程建筑、直接出兵、精兵、法术和战鼓。
- `万灵召使`：更多生产建筑和防守牌，直接出兵较少，偏持续铺场。
- `雷火方士`：建筑更少，更多法术、控制和费用节奏牌，偏主动清场。

### 3. 职业被动差异

新增职业规则：

- `边境指挥官`
  - 士兵和精兵攻击略高。
  - 士气节奏保持标准 5 个士兵触发 1 点。
- `万灵召使`
  - 阵位更多。
  - 建筑牌费用更低。
  - 建筑产兵更快。
  - 士气触发更早。
  - 法术牌更贵，费用回复略慢。
  - 最大生命更高。
- `雷火方士`
  - 法术与减益牌费用更低。
  - 法术伤害更高。
  - 费用上限、开局费用、手牌上限更高。
  - 费用回复更快。
  - 建筑牌更贵，建筑产兵更慢，阵位略少。
  - 最大生命更低。

### 4. 职业奖励倾向

卡牌奖励和商店抽卡现在会根据职业调整权重：

- `万灵召使` 更容易看到建筑、士兵、战术牌。
- `雷火方士` 更容易看到法术、减益、经济牌。
- `边境指挥官` 更偏士兵、精兵、战术牌。

这一步让路线奖励更容易沿着职业方向形成流派。

### 5. 职业神通

同一个战斗神通按钮会根据职业释放不同效果：

- `边境指挥官`：边境战令，对敌军造成范围伤害，给前线加盾并获得士气。
- `万灵召使`：万灵开阵，立刻召来一批援军并推进士气。
- `雷火方士`：九霄雷火，对全场敌人造成更高雷火伤害，并短暂定身非首领单位。

### 6. 英雄牌死亡规则

英雄牌现在有完整唯一性规则：

- 同名英雄存活时不能重复召唤。
- 英雄本场战斗阵亡后，后续即使洗回卡池也不能再次召唤。
- 手牌会显示具体原因：`该英雄已经在场` 或 `该英雄本场已阵亡`。

### 7. 内容校验扩展

MVP 校验现在会检查 3 个职业的初始卡组：

- 初始卡组卡牌数量不能过少。
- 初始卡组中的卡牌 ID 必须存在。
- 每个职业至少要有生产建筑，避免卡组无法支撑核心战斗。

### 8. 迷宫固定房间与连线规则

- 每层第 1 个房间固定为单个普通怪物房间。
- 每层第 5 个房间固定为单个精英房间。
- 每层第 10 个房间固定为单个首领房间。
- 普通推进房间仍然保持 2-3 个。
- 单个房间不会再连接上方全部房间；没有连线的房间会保持不可走状态，路线选择更有取舍。
- MVP 校验同步检查固定房间数量和连线是否过度展开。
