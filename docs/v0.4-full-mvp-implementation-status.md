# v0.4 完整 MVP 开发记录

## 本次目标

把第一版 MVP 从“能连续战斗”推进到“完整肉鸽探索可测”：商店能真正调整卡组，最终首领有阶段压力，主角经验和永久神器能影响后续探索。

## 已完成功能

### 1. 商店闭环

- 商店进入时生成固定货架，购买后该商品会从当前货架移除。
- 新增货架刷新，刷新价格随次数递增，并受 `通宝令` 折扣影响。
- 新增卡牌净化：每次商店可花金币移除 1 张已有卡牌，卡组过薄时禁止继续移除。
- 保留半价出售已有卡牌。

### 2. 局外成长

- 主角总经验会计算主角等级。
- 主角等级会给后续探索提供少量初始金币和生命上限。
- 永久神器 `通关遗珍` 会继续提供初始金币和生命收益。
- 标题界面、探索顶部栏、结算界面都会显示永久探索次数、通关次数、总经验、主角等级等信息。

### 3. 最终首领阶段

- 最终首领低于 70% 血量时进入二阶段：额外召唤敌军，并提前释放首领技能。
- 最终首领低于 40% 血量时进入三阶段：压制费用回复、扣除部分费用、冲击玩家建筑。
- 首领战仍然保持“核心站桩输出 + 派兵 + 技能”的规则，不会直接攻击玩家基地。

### 4. 卡牌和状态补充

- 新增 `缚妖索符`：目标区域减速并短暂定身。
- 新增 `甘霖咒`：治疗并保护前线单位。
- 新增 `流云军令`：抽牌并回复费用，用于调整手牌节奏。
- 新增 `破阵金令`：削弱敌方前线攻击。
- 战斗单位已支持减速和眩晕类状态。
- 士气现在可以强化不自带士气生成的战术牌，`战鼓法令` 不会因为自己生成士气而误吃已有士气。

### 5. 测试入口

- 迷宫界面新增小型调试条：
  - `+100 金`
  - `回满血`
  - `给神器`
  - `到首领`
  - `最终首领`
- 战斗界面保留快速胜利、失败、奖励、士气等调试按钮，方便测试三层通关闭环。

## 当前验收状态

- 3 层 10 行迷宫、路线选择、节点奖励、商店、休息、机遇、神秘、神器、小首领、最终首领均已接入。
- 当前版本已经具备第一版 MVP 的完整探索闭环。
- 后续重点应转向数值调优、战斗手感、美术资源一致性、WebGL 字体与打包资源稳定性。
